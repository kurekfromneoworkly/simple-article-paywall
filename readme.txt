Simple paywall system for blog posts with WooCommerce integration. Allows selling access to individual articles.

== Description ==

Simple Article Paywall enables you to monetize your content by restricting access to full articles. Visitors can read a preview of the article and purchase 24-hour access to read the complete content.

**Features:**

* Easy to set up and use
* Individual article restrictions
* WooCommerce integration
* Customizable preview length
* 24-hour access period
* Automatic product creation
* Works with any WordPress theme

**Requirements:**

* WordPress 5.0 or higher
* WooCommerce 3.0 or higher
* PHP 7.2 or higher

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/simple-article-paywall`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure WooCommerce is installed and activated
4. Go to any post and enable the paywall in the sidebar

== Frequently Asked Questions ==

== Privacy Policy ==

Simple Article Paywall uses WooCommerce to process payments and manage access to restricted content. It stores the following data:
* Purchase history for article access
* User access permissions
* Payment information (handled by WooCommerce)

No additional personal data is collected or stored by this plugin.
